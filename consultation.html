OpSec Consultation

Level 1 - $200(AUD)

The one-hour introductory digital security health check covers key areas of online safety. It includes reviewing social media accounts with tools like Have I Been Pwned, emphasising strong passwords, password managers, and two-factor authentication (2FA). Account security involves adjusting settings and monitoring for unauthorised access. Participants will update systems and applications, secure Wi-Fi with WPA3 encryption, use VPNs, and configure firewalls. The session focuses on regular, secure backups, managing privacy settings, auditing app permissions, and promoting secure browsing practices. Additionally, it will cover physical security and provide hardware recommendations for enhanced security.
